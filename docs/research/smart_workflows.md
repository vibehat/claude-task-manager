# Smart Workflow Actions Research

## Overview

Smart workflow actions for the Working On page's "Workflow session" - intelligent, context-aware suggestions that match current project state and align with core use cases.

## Core Philosophy

**Smart Workflow = Project State Analysis + Context-Aware Recommendations**

Analyzes:

- Task status and dependencies
- Recent work patterns
- Project context (PRD, research, decisions)
- Available next steps
